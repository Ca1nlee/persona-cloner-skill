# Worldview Map Reference

Map how the target sees reality, not just what opinions they once stated.

## Required sections

### 1. Core beliefs
List stable beliefs about:
- people
- institutions
- work
- truth / knowledge
- risk
- success / failure
- morality / responsibility
- time and leverage

### 2. Value hierarchy
Rank what the person appears to optimize for.
Include explicit tradeoffs.

### 3. Lens library
Document the lenses they repeatedly use:
- first-principles
- incentives
- systems
- craft
- status
- moral framing
- competition / strategy
- human nature

### 4. Non-negotiables
List beliefs or values that repeatedly override convenience.

## Rule

A worldview map should predict future judgments in adjacent situations.
If it cannot, it is probably still a quote collection.

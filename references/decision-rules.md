# Decision Rules Reference

Translate worldview into operational decisions.
A good persona package must make choices in a way that resembles the target's prioritization logic.

## Required sections

### 1. Default heuristics
Document repeated decision shortcuts.
Examples:
- when uncertain, shrink scope
- prefer direct evidence over abstraction
- choose speed over polish until threshold X

### 2. Tradeoff system
For each major tradeoff, define:
- what wins by default
- what evidence can reverse the preference
- what failure mode appears when over-applied

### 3. Thresholds
Document where the target changes behavior:
- evidence thresholds
- trust thresholds
- scale thresholds
- quality thresholds
- risk thresholds

### 4. Stress behavior
Document how decisions shift under:
- time pressure
- conflict
- ambiguity
- reputational risk
- resource scarcity
